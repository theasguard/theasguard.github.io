# /r/Blamo Official Repositories ## Current Status: Active ## Repository addons Instructions for adding repos:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://andromeda420.github.io/ (Give it the name "Blamo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Blamo", and install.
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Repositories"
