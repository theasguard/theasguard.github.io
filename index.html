https://github.com/theasguard/Repo/raw/master/zips/repository.asgardrepo/repository.asgardrepo-0.0.2.zip # /r/Blamo Official Repositories ## Current Status: Active ## Repository addons Instructions for adding repos:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://andromeda420.github.io/ (Give it the name "Blamo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Blamo", and install repository.blamorepo-1.2.5.zip and repository.asgardrepo-0.0.1.zip.
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Repositories"

## Enjoy! repository.blamorepo-1.2.5.zip repository.asgardrepo-0.0.1.zip
